## What I Learned

The Robot Simulator kata involves programming a robot to move on an infinite 2D grid based on a string of commands:

| Command | Action |
|---------|--------|
| `A` | Advance one step in the current direction |
| `L` | Turn 90° to the left |
| `R` | Turn 90° to the right |

The robot tracks its **position** `(x, y)` and **direction** `(N, S, E, W)`.

---

## Key Concepts Implemented

### 1. Directional Turning
Directions are stored in a circular clockwise list: `['N', 'E', 'S', 'W']`
- **Turn Right** → index + 1 (mod 4)
- **Turn Left** → index - 1 (mod 4)

### 2. Advancing
Each direction maps to an (x, y) delta:
- `N` → `(0, +1)` · `S` → `(0, -1)` · `E` → `(+1, 0)` · `W` → `(-1, 0)`

### 3. Command Execution
A loop iterates over each character in the command string and dispatches to the correct method.
